JS实现选项卡功能

更新：2016-01-27

如需转载请以明文链接形式注明出处

下图的选项卡功能在网站中随处可见，今天就让我们来看一下如何用JS实现这种简单的选项卡功能。

我们先用HTML和CSS来完成基本的布局

<style> #chooseTab div{ width:276px; height:200px; background:#DDD; color:white; display:none; } #chooseTab input{ padding:3px 7px; border:none; } .active{ background:blue; color:white; } </style> <body> <div id="chooseTab"> <input type="button" value="选项卡1" class="active" /> <input type="button" value="选项卡2" /> <input type="button" value="选项卡3" /> <input type="button" value="选项卡4" /> <div style="display:block">选项卡1</div> <div>选项卡2</div> <div>选项卡3</div> <div>选项卡4</div> </div> </body>

然后用JS来实现选项卡的功能

<script> window.onload = function(){ var aBtn = document.getElementById('chooseTab').getElementsByTagName('input'); var aDiv = document.getElementById('chooseTab').getElementsByTagName('div'); for (var i = 0; i < aBtn.length; i++) { aBtn[i].index = i; aBtn[i].onclick = function(){ for (var i = 0; i < aBtn.length; i++) { aBtn[i].className = ""; aDiv[i].style.display = "none"; }; this.className = "active"; aDiv[this.index].style.display = "block"; }; } } </script>

当然，这只是最简单的实现方法，ui设计得比较简洁，甚至可以说是简陋，但是麻雀虽小五脏俱全，其代码还是非常好用的。
快来实现你自己的选项卡吧。
