让IE兼容background-size的方法

更新：2016-02-10

background-size 在ie下使用

background-size属性规定背景图像的尺寸，但是在IE6、IE7以及IE8中并不兼容，如果想要在它们中使用background-size该如何实现呢？前几天我在学习这一块的时候 其实方法很简单，只需要将下面的代码添加到CSS中就可以了：

filter: progid:DXImageTransform.Microsoft.AlphaImageLoader( src='图片链接', sizingMethod='scale');

下面看一个实例

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Document</title> <style> div{ width:280px; height:176px; background-image:url("http://i13.tietuku.com/627ed5c8b3adbba6.jpg"); background-size:cover; filter: progid:DXImageTransform.Microsoft.AlphaImageLoader( src='http://i13.tietuku.com/627ed5c8b3adbba6.jpg', sizingMethod='scale'); } </style> </head> <body> <div> </div> </body> </html>
